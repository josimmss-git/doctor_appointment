# 🩺 DocAppoint - Doctor Appointment Manager

A modern doctor appointment booking platform where patients can browse doctors, view detailed profiles, book appointments, and manage their bookings securely. The application includes JWT-based authentication, profile management, appointment management, and a responsive user-friendly interface.

## ✨ Features

* Browse and explore available doctors with detailed information.
* Secure authentication using Better Auth with JWT/Session support.
* Book doctor appointments and save booking data to MongoDB.
* Manage appointments with update and delete functionality.
* Personal dashboard with My Bookings and My Profile sections.
* Search doctors by name from the All Appointments page.
* Responsive design optimized for mobile, tablet, and desktop devices.
* Dynamic doctor details page with appointment booking system.
* Real-time toast notifications for all CRUD operations.
* Custom 404 page and loading spinner for improved user experience.
* SEO-friendly metadata implementation for all pages.
* Protected routes to secure user-specific data.
